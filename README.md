## Backup & Restore Automation System

## 📌 Description
This project automates backup and restore operations using Bash scripting and cron jobs in Linux.

## ⚙️ Features
- Automated scheduled backups
- Compression using tar
- Logging system
- Restore functionality
- Backup retention policy

## 🛠️ Technologies
- Linux
- Bash Scripting
- Cron Jobs

## 🚀 Usage

### Run Backup
./backup.sh

### Restore Backup
./restore.sh <backup_file> <restore_directory>

## ⏰ Cron Example
0 2 * * * /path/to/backup.sh
